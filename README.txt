This mod was made for minecraft using Forge 1.12.2-14..25.5.2768
This Project contains a copy of the mod as well as full source code required to rebuild the mod. 

Maxium added Blocks, Tools, Weapons, Armor, and Blocks for two new item sets. 
    - The Mythic Wood items had similar durability to Iron but with the enchantability of Gold. 
    - Mythril had durability higher than Iron but lower than Diamond. It also had the enchatability of Gold.
    - There are numerous other decrotive purple/pink blocks added for building purposes. 
    - Many of the blocks have a glow to them to help keep mobs away.
